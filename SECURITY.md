# Security Policy

## Supported Versions

Only the latest active semantic release version receives security patches.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please report security issues or leaks privately by emailing security@aieos.org. We aim to respond within 48 hours to coordinate a private disclosure path.
