# Critical Thinking Capability Contract

## Core Purpose
Detect cognitive biases, identify hidden assumptions, and challenge decision frameworks.

## Quality Gates
- **Entry Preconditions**: Draft plan or decision log file
- **Required Context**: Goal parameters and historical project outcomes
- **Execution Instructions**: Trace premises to evidence, search for contradictions, and challenge variables
- **Verification Assertions**: All assumptions are categorized by confidence levels
- **Exit Requirements**: Verified critical review document
