# Alternative Exploration Capability Contract

## Core Purpose
Map alternative paths, evaluate simpler/cheaper approaches, and avoid lock-in.

## Quality Gates
- **Entry Preconditions**: Core project requirements specification
- **Required Context**: Budget and scalability boundaries
- **Execution Instructions**: Model 3 distinct approaches and map cost/migration curves
- **Verification Assertions**: At least one alternative has a significantly lower complexity footprint
- **Exit Requirements**: Completed alternatives matrix markdown
