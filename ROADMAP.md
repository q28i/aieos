# AIEOS Release Roadmap

## v1.3.0 (Current - Clean Workspace Release)
- Overhauled workspace layout to scaffold all environment directories inside `.aieos/` folder.
- Separated global runtime assets and packages (`~/.aieos/`) from local project-specific decision memory.
- Added `uninstall` route/command to cleanly remove integrations and global runtimes while leaving project decision history intact.
- Introduced SaaS, Computation Engine, Backend, Library, AI Agent templates in the installation wizard.

## v1.2.0 (Stable Release)
- Added Socratic interactive prompt installer console wizard.
- Added Architect, Mentor, Reviewer, and Decision OS behavior profiles.

## v1.1.0 (Stable Release)
- Replaced commands with direct, zero-config installers for AI agent environments.
- Added bundle flag mapping capability domains.
